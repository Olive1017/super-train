# 万益特货品装箱计算器

基于 Streamlit 开发的货品装箱计算工具，支持多种货品混装，自动生成最优装箱方案和可视化图表。

## 功能特点

- 🚚 支持 2-3 种货品混装
- 📦 自动计算最优装箱方案
- 🎯 智能分段布局：支持2-3段布局，同种货品可分段装载
- 🔄 混合方向摆放：同一段内可混用横竖向，最大化空间利用
- 📊 完整层数搜索：遍历所有可能的层数组合，确保最优解
- 📐 俯视图可视化展示
- ⚖️ 确保前后堆高差 ≤ 3cm（理想）或 ≤ 10cm（台阶式）
- 🎨 简洁美观的界面

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中自动打开，默认地址为 `http://localhost:8501`

## 部署到云端

### Streamlit Cloud（推荐，免费）

1. 将代码推送到 GitHub
2. 登录 [Streamlit Cloud](https://streamlit.io/cloud)
3. 点击 "New app"，连接您的仓库
4. 选择 `app.py` 作为主文件
5. 点击 "Deploy"
